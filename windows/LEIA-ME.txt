JARVIS - agente de terminal (Windows)
=====================================

PRE-REQUISITO
  Python instalado com "Add python.exe to PATH" marcado.
  (baixe em https://python.org)

COMO INSTALAR
  1. Copie esta pasta inteira para a maquina Windows.
  2. De dois cliques em  INSTALAR.bat
     (ele cria o ambiente, instala as dependencias e o comando 'jarvis').
  3. Abra um NOVO terminal (cmd) e digite:  jarvis

ONDE FICA CADA COISA
  - Programa:          %LOCALAPPDATA%\jarvis
  - Chaves da API:     %APPDATA%\jarvis\chaves.txt   (ja vem preenchido)
  - Arquivos gerados:  Documentos\JARVIS   (planilhas, PDFs, etc.)

USO
  jarvis                 abre o chat
  jarvis "sua tarefa"    pergunta rapida

DESINSTALAR
  De dois cliques em  DESINSTALAR.bat

OBS
  As chaves da API do Gemini ja vem embutidas no arquivo chaves.txt.
  O free tier pode usar seus prompts para treino: nao coloque dados sensiveis.
